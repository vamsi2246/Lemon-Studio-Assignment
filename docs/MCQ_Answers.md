Q1. Which of the following is the primary purpose of embeddings in a Retrieval-Augmented Generation (RAG) pipeline? A1. The main purpose of embeddings is to convert text into numerical vector representations that capture semantic meaning, enabling the system to retrieve contextually relevant documents even when exact keywords don't match the query.

Q2. Which HTTP method is most appropriate for creating a new resource through a REST API? A2. POST is the correct method. In RESTful conventions, POST is specifically designed to submit data and create new resources on the server, unlike GET (read), PUT (update), or DELETE (remove).

Q3. In React, which concept is primarily used to manage component-level dynamic data? A3. State is used to manage dynamic data within React components. When state changes, React automatically re-renders the component to reflect the updated values in the UI.

Q4. What is one major advantage of using FastAPI or Express.js for backend services? A4. They significantly reduce development time by providing built-in routing, middleware support, and request handling out of the box. FastAPI additionally offers automatic API documentation and native async support for better performance.

Q5. In a Gen-AI application, hallucination refers to: A5. Hallucination refers to when an AI model generates responses that sound convincing and well-structured but contain fabricated or factually incorrect information not supported by any source data.
