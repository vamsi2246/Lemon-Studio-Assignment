# MCQ Answers

## Q1. What is the primary purpose of embeddings in a RAG pipeline?

### Answer:

To convert text into vectors (numbers) so similar content can be found easily.

### Simple Explanation:

Embeddings help the system understand the meaning of text and find relevant information even if the exact words are different.

---

## Q2. Which HTTP method is used to create a new resource in a REST API?

### Answer:

POST

### Simple Explanation:

POST is used when sending new data to the server, such as creating a user or uploading a document.

---

## Q3. In React, what is used to manage component-level dynamic data?

### Answer:

State

### Simple Explanation:

State stores data that can change over time and updates the UI automatically when the data changes.

---

## Q4. What is one major advantage of using FastAPI or Express.js?

### Answer:

They make backend development faster and easier.

### Simple Explanation:

These frameworks provide routing, API handling, and other useful features, reducing development time.

---

## Q5. What is hallucination in a Gen-AI application?

### Answer:

When an AI generates incorrect information as if it were correct.

### Simple Explanation:

The AI gives an answer that sounds convincing but is actually wrong or made up.
